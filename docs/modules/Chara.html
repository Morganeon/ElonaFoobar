Chara module

Functions for working with characters.

Usage

local Chara = Elona.require("Chara")

Functions

is_alive (character) Checks if a character is alive.
is_player (character) Checks if a character is the player (has index 0).
is_ally (character) Checks if a character is a member of the player's party (has index < 16 and != 0).
player () Returns a reference to the player.
count () Returns the number of other characters in the current map.
flag (chara, flag) Gets a flag on a character.
create (position, id) Attempts to create a character at a given position.
iter (from, to) Iterates between two character indices.

Enums module

Enumerations for various data types.

Usage

local Enums = Elona.require("Enums")

Enums

IdentifyState The identification state of an item.
CurseState The curse state of an item.
StatusAilment A status effect applicable to a character.
TileKind A type of tile to be used with Map.generate_tile.
DamageSource A source of damage to be used with LuaCharacter.damage_hp.
CharaFlag A flag that can be set on a character.

# IdentifyState

The identification state of an item.

Enum Values:
  • Unidentified : (Enums.IdentifyState)
  • Partly : (Enums.IdentifyState)
  • Almost : (Enums.IdentifyState)
  • Completely : (Enums.IdentifyState)
Usage:

item.identify_state = Enums.IdentifyState.Completely

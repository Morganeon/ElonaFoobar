Item module

Functions for working with items.

Usage

local Item = Elona.require("Item")

Functions

count () Returns the number of items on the ground.
create (position, id, number) Attempts to create an item of the given quantity at a position.
has_enchantment (item, id) Checks if an item has an enchantment.
remove (item) Removes this item from the game world.
iter (from, to) Iterates between two item indices.

Functions

# count ()

Returns the number of items on the ground.

Returns:
  • (num) the item count
# create (position, id, number)

Attempts to create an item of the given quantity at a position.

Returns the item stack if it was created, nil otherwise.

Parameters:
  • position : (LuaPosition) (const) position to create the item at
  • id : (num) the item prototype ID
  • number : (num) the number of items to create
Returns:
  • (LuaItem) the created item stack
Or
  • (nil)
# has_enchantment (item, id)

Checks if an item has an enchantment.

Parameters:
  • item : (LuaItem) (const) an item
  • id : (num) the ID of the enchantment
Returns:
  • (bool) true if the item has the enchantment
# remove (item)

Removes this item from the game world.

The item passed in will no longer be valid for use.

Parameters:
  • item : (LuaItem) (mut) an item
# iter (from, to)

Iterates between two item indices.

Parameters:
  • from : (num) the starting index
  • to : (num) the ending index
Returns:
  • (iterator) an item iterator
